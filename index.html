How It Works

This tool converts text to invisible zero-width characters and back again:

  • Zero-width space (U+200B): An invisible character used for hidden text encoding
  • Zero-width joiner (U+200D): Another invisible character used in the encoding process
  • Zero-width non-joiner (U+200C): Used as a separator in the encoded text
  • Conversion process: Text is converted to binary, then to invisible characters
  • Reversal: The process is reversed to decode invisible text back to readable form

Use cases: Hide messages in plain text, encode secret data, or test how systems handle invisible characters.
